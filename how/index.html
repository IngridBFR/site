How It Works

Convox runs your infrastructure so that you can focus on your applications.
Get Started For Free

Configure, develop, and deploy.

One configuration file to describe your application.

version '2'
services
web
build .
environment
- NODE_ENV=development
links
- redis
ports
- 80:8000
redis
image convox/redis
ports
- 6379

One command to boot your app on any Windows, Mac or Linux laptop.

convox start
build │ Step 1: FROM ubuntu 16.04
build │ Step 2: RUN apt-get install nodejs npm
build │ Step 3: RUN npm install
redis │ The server is now ready on port 6379
web │ Web running on port 8000

One command to deploy your application to production.

convox deploy
